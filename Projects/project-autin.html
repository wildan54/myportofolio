AUTIN – Autism Independence Monitoring System

Python (Flask) Bootstrap

Overview

AUTIN is a web-based system designed to analyze and monitor the independence level of children with autism using video-based activity recognition. The system evaluates user-submitted videos and classifies behavior into categories such as verbal, gesture, and physical assistance to determine independence levels.
